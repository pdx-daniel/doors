# Guides

Suggested guides for contributors to this project. Each should be a standalone markdown file in this directory.

| # | Guide | Covers |
|---|-------|--------|
| 1 | **Local setup** | First-time bootstrapping — dependencies, environment, database, and running every target |
| 2 | **Database** | Schema migrations, seeding, connecting via CLI, and adding a new table |
| 3 | **API routes** | Adding a new Elysia route, middleware, request validation, and Eden Treaty type generation |
| 4 | **Basemap tiles** | Generating PMTiles with Planetiler, hosting options, and swapping regions |
| 5 | **Map integration** | MapLibre setup on native vs. web, cluster layers, viewport queries |
| 6 | **Navigation & screens** | Adding a tab, a stack screen, linking, and shared layout patterns |
| 7 | **Workspace isolation** | How the `X-Workspace-Id` header scopes data, multi-tenant patterns, and testing |
| 8 | **Web build & deploy** | Webpack config, production build, static hosting, and SPA routing |
| 9 | **Mobile build & deploy** | iOS/Android release builds, code signing, and CI distribution |
| 10 | **Quality** | Biome config, TypeScript strictness, docstring conventions, and CI pipeline |
